## 🆘 Need Help?

### Common Issues:

**Q: npm install fails**
A: Make sure Node.js 14+ is installed. Run `node --version` to check.

**Q: Music doesn't play**
A: Add your music file to `public/music/wedding-music.mp3`. Some browsers block autoplay (this is normal).

**Q: Images don't show**
A: Check file paths in `src/components/Gallery.js`. Use `/images/photo.jpg` format.

**Q: Maps don't load**
A: Verify your internet connection and Google Maps embed URLs.

## 💡 Pro Tips

### Development:
1. Keep the terminal open to see errors
2. Save files to see changes instantly
3. Use browser DevTools (F12) for debugging
4. Test on mobile using browser DevTools
5. Keep backups of your changes

### Customization:
1. Make one change at a time
2. Test after each change
3. Keep the color scheme consistent
4. Don't remove Islamic elements
5. Maintain responsive design

### Deployment:

2. Use production build (`npm run build`)
3. Test the live site thoroughly
4. Enable HTTPS for security
